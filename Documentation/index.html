How To Install Blogger Template

If you are here, probably you want to install this template on your blogger blog. You need to have a free blogger blog (account) to proceed. The tutorial for that is out of scope of this documentation. Please GOOGLE.

  • Go To Blogger Dashboard
  • On Blogger Dashboard click Template
  • Now Click On Backup/Restore button (Top Right).
  • Click Choose File button. Find where the “wecommerce.xml” file location.
  • Now Click On Upload Button.

About Payouts And Checkout Pages.

Go to Template Editor And Locate The Very Top Script. Change It To Your Email Address For Paypal Payouts To Your Email Address.

Also, Checkout Page Is Restricted To /p/cart.html. You cannot change the name in free version. Go here and copy everything you see and paste it in HTML mode of a New Page in your blogger blog. Make sure the URL of the page is cart.html.

Create a page with URL /p/thank-you-for-purchase.html and add anything you want to appear to user after the successful payment. Make sure to keep the name intact.
